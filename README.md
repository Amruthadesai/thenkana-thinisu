# Thenkana Thinisu · ತೆಂಕಣ ತಿನಿಸು

A single-page website for **Thenkana Thinisu**, a homemade South Indian snacks business
from Coastal Karnataka. Orders are taken over WhatsApp / phone.

## Structure

- `index.html` — page markup (Home, Story, Menu, Gallery, Contact)
- `css/style.css` — styles (responsive, mobile nav, scroll reveal)
- `js/script.js` — mobile menu toggle, scroll-reveal animation, footer year
- `images/snacks/` — snack photos

## Images used

Snacks (`images/snacks/`): `hero`, `kajjaya`, `nippattu`, `kodubale`, `chakli` (`.jpg`).
Replace any placeholder photos with real ones using the same filenames.
To add more snacks, drop the image in `images/snacks/` and add a matching `.card` in `index.html`.

Coastal Karnataka is represented with lightweight CSS/SVG decorative bands (no image files):
the mango-leaf **toran**, **kolam** heading ornaments/divider, a beach **wave** divider,
and a **Yakshagana** crown divider — see `css/style.css`.
